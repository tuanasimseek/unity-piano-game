# Piyano Oyunu — Unity

Unity ile geliştirilmiş, oyuncuların düşen notalara senkronize şekilde tuşlara bastığı müzik tabanlı bir piyano oyunu.

## Özellikler
- Müzikle senkronize düşen nota sistemi
- Ses geri bildirimi ile piyano tuş etkileşimi
- Skor takibi

## Kullanılan Teknolojiler
- Unity
- C#

## Nasıl Çalıştırılır
1. Projeyi Unity Hub ile açın
2. Editörde Play tuşuna basın

## Proje Yapısı
- `Scripts/` — Oyun mantığı ve input yönetimi
- `Prefabs/` — Yeniden kullanılabilir oyun objeleri
- `Sounds/` — Ses dosyaları
